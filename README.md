# Flutter Chat Application with Firebase (CRUD & Authentication)

## Overview
A real-time chat application built with *Flutter* and *Firebase* that supports user authentication, CRUD operations on messages, and responsive chat UI for both Android and iOS platforms. This app uses *Firebase Authentication* for secure login and *Firestore* for real-time message storage and retrieval.

## Features
- *User Authentication*: Sign up, login, and logout with Firebase Authentication.
- *Real-time Messaging*: Exchange messages in real-time using Firebase Firestore.
- *CRUD Operations*: Create, Read, Update, and Delete messages within the chat.
- *User Profiles*: Edit user profile information such as display name and status.
- *Responsive UI*: Works seamlessly across Android and iOS devices.
- *Push Notifications*: (Optional) Push notifications for new messages using Firebase Cloud Messaging (FCM).
  
## Technologies Used
- *Flutter*: Cross-platform framework for mobile app development.
- *Firebase Firestore*: Real-time NoSQL cloud database for storing chat messages.
- *Firebase Authentication*: Secure user authentication for the app.
- *Firebase Cloud Messaging (FCM)*: For push notifications.
- *State Management*: Provider or GetX (depending on your choice).


## Getting Started

### Prerequisites
Before you begin, ensure you have met the following requirements:
- Flutter SDK installed on your machine.
- A Google Firebase account.

### Installation


    

1. *Set up Firebase*:
    - Go to the [Firebase Console](https://console.firebase.google.com/).
    - Create a new Firebase project.
    - Enable Firebase Authentication (Email/Password).
    - Enable Firestore Database.
    - Download the google-services.json file (for Android) or GoogleService-Info.plist (for iOS) and place them in the corresponding directories (android/app and ios/Runner respectively).
